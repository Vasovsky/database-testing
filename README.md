# Database Testing Project

Этот проект предназначен для демонстрации моих навыков в области тестирования баз данных. В рамках проекта я создал простую базу данных SQLite и написал тесты, которые проверяют целостность данных и правильность SQL-запросов. Тесты включают в себя проверки на количество записей, уникальность значений в столбцах, а также проверку конкретных данных в базе.

Основная цель проекта — научиться и продемонстрировать умение работать с базами данных, писать и выполнять тесты для проверки их целостности. Проект включает базовую настройку базы данных, а также тестирование базовых операций с данными.

## Используемые технологии

- **Python 3.10+**: Основной язык программирования.
- **SQLite**: Легковесная база данных, используемая для хранения данных.
- **unittest**: Встроенный модуль Python для создания и выполнения тестов.

## Как запустить проект

1. **Установка Python**: Убедитесь, что у вас установлен Python версии 3.10 или выше. Если Python не установлен, вы можете скачать его с [официального сайта Python](https://www.python.org/downloads/).

2. **Клонирование репозитория**: Клонируйте этот репозиторий на свой компьютер:
    ```bash
    git clone https://github.com/Vasovsky/database-testing.git
    cd database-testing
    ```

3. **Создание и активация виртуальной среды**: Создайте виртуальную среду, чтобы изолировать зависимости проекта:
    ```cmd
    python -m venv venv
    venv\Scripts\activate
    ```

4. **Создание базы данных**: Запустите скрипт для создания базы данных и заполнения её начальными данными:
    ```bash
    python db\database_setup.py
    ```

5. **Запуск тестов**: После создания базы данных выполните тесты, чтобы убедиться, что все работает корректно:
    ```bash
    python -m unittest discover tests
    ```

## Как это работает

В проекте реализованы тесты, которые проверяют следующие аспекты работы с базой данных:

1. **Количество записей в таблице**: Проверка, что в таблице `users` содержится 3 записи.
2. **Уникальность email-адресов**: Проверка, что все email-адреса в таблице уникальны.
3. **Проверка данных пользователя**: Проверка, что данные для пользователя с именем "Alice" совпадают с ожидаемыми.
